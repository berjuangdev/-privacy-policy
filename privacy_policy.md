# Kebijakan Privasi - To-Do App

**Terakhir diperbarui:** Juni 2026

## 1. Pendahuluan

To-Do App ("aplikasi") dikembangkan sebagai aplikasi produktivitas untuk membantu pengguna mengelola tugas harian. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda.

---

## 2. Data yang Kami Kumpulkan

### Data Akun
- Alamat email yang Anda gunakan untuk mendaftar
- Data ini dikelola oleh **Firebase Authentication** (Google)

### Data Tugas
- Judul dan catatan tugas yang Anda buat
- Tanggal dan waktu tugas
- Status penyelesaian tugas
- Data ini disimpan di **Cloud Firestore** (Google) dan di perangkat Anda secara lokal

### Data Kamera & Galeri
- Aplikasi meminta izin kamera dan galeri **hanya** untuk fitur scan teks (OCR)
- Gambar yang Anda ambil atau pilih **tidak disimpan** di server manapun
- Gambar diproses langsung di perangkat menggunakan Google ML Kit

---

## 3. Bagaimana Kami Menggunakan Data

Kami menggunakan data Anda **hanya** untuk:
- Menampilkan dan menyimpan tugas Anda
- Menyinkronkan data antar perangkat melalui akun Anda
- Mengidentifikasi akun Anda saat login

Kami **tidak**:
- Menjual data Anda kepada pihak ketiga
- Menggunakan data Anda untuk iklan
- Membagikan data Anda kepada siapapun

---

## 4. Penyimpanan Data

- Data disimpan di server **Google Firebase** yang berlokasi di **asia-southeast2 (Jakarta)**
- Data lokal disimpan di perangkat Anda menggunakan Hive
- Anda dapat menonaktifkan penyimpanan online kapan saja melalui menu aplikasi

---

## 5. Keamanan Data

- Setiap pengguna hanya dapat mengakses data miliknya sendiri
- Data dilindungi dengan Firebase Security Rules
- Koneksi menggunakan enkripsi HTTPS/TLS

---

## 6. Hak Pengguna

Anda berhak untuk:
- **Mengakses** data Anda kapan saja melalui aplikasi
- **Menghapus** tugas Anda kapan saja
- **Menghapus akun** Anda dengan menghubungi kami
- **Menonaktifkan sinkronisasi** online kapan saja
